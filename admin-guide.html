Ugarit Admin Guide

Manual user creation and management system

How to Create a New User

Open browser console (F12) on any page and run:

ugaritAdmin.createUser({ walletId: "john_doe123", password: "securepassword123", fullName: "John Doe", phone: "+963123456789", email: "john@email.com", username: "john_doe" });

Available Admin Commands

List All Users

ugaritAdmin.listUsers()

Update User Balance

ugaritAdmin.updateBalance("john_doe123", 1000)

Add Transaction

ugaritAdmin.addTransaction("john_doe123", { type: "deposit", amount: 500, status: "completed", description: "Bank transfer deposit" });

Get User Info

ugaritAdmin.getUser("john_doe123")

Delete User

ugaritAdmin.deleteUser("john_doe123")

User Data Structure

{ walletId: "unique_wallet_id", password: "user_password", userData: { fullName: "User Full Name", phone: "Phone Number", email: "user@email.com", username: "preferred_username" }, balance: 0, currency: "SYP", status: "active", createdAt: "2024-01-01T00:00:00.000Z", lastLogin: null, transactions: [] }
